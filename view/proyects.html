BonAreaIA

November 2020

The system works through a neural network called Yolo v3 which checks the images in real time and labels them looking for holes in the shelves of refrigerators. It saves the coordinates of these gaps, and then sends this information to the mobile application. The application reads the data received and counts how many holes there are and in which area of ​​the refrigerator they are. It then notifies the user of any product deficiencies in the refrigerator, how much, and in what area. Because the images are analyzed in real time, if an employee receives a notification of a gap and fills it, the notification will automatically disappear in the application and mark the refrigerator as full again.

Pokemon2Digimon

December 2021

In this repository you will find a classifier made with neural networks. The themes I touch on are:

  • I check if the proposed models are capable of solving this classification problem.
  • The type of drawing of the images is analyzed, such as what parts of the image represents a pokemon or a digimon.
  • I create a pokemon that maximizes the classifier's "Pokemon" class.
  • I create a Digimon that maximizes the "Digimon" class of the classifier.
  • I check how resistant the best model is, using adversary attacks.
